A Binary file converter for Super Nintendo development  
Developed by Alekmaul and distributed under the terms of the [MIT license](./LICENSE).

## Usage
```
bin2txt [options] filename  
```
where filename is a binary file  

## Options
### Convertion options  
- `-cc` output in c style format  
- `-ca` output in assembly style format  

### Misc options  
- `-h` display help  
- `-q` quiet mode  
- `-v` display version information
